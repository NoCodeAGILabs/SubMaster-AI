# SubMaster-AI  
Subtítulos karaoke virales 100% offline – hecho con prompts

úsalo, véndelo, rómpelo, hazte rico. Me da igual mientras funcione.

- 100% local (nada de nube)  
- Efectos virales tipo TikTok/Instagram  
- Funciona en cualquier PC con Python  
- Hecho sin saber programar, solo con 4000+ prompts

## Cómo usarlo (30 segundos)
1. `pip install -r requirements.txt`
2. `python karaoke_generator.py`
3. Arrastra tu vídeo → elige efecto → exporta → a reventar redes
